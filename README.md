**TeamTrack** – Employee Attendance & Leave Dashboard

TeamTrack is a smart **Employee Attendance & Leave Analytics Dashboard** built with **Python & Streamlit**. It helps employees and managers track attendance, leaves, and performance insights interactively.

## **Features**

* **Employee Login**: Check attendance %, leave balance, and performance insight.
* **Admin Dashboard**: View all employees, charts, trends, and low attendance alerts.
* **Analytics**: Attendance charts, task ratings, and real-time insights.
* **Downloadable Reports**: Export CSV of attendance and leave data.



## **Tech Stack**

**Python, Streamlit, Pandas, Matplotlib, CSV**


## **Folder Structure**

```
TeamTrack/
├── app/       → Streamlit dashboard code
├── data/      → employee datasets (CSV)
├── assets/    → screenshots
├── README.md
├── requirements.txt
```

---

## **Run Locally**

```bash
pip install -r requirements.txt
streamlit run app/app.py
```

Open in browser: `http://localhost:8501`
